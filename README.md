# Curiosity Institute — Local Website

Simple responsive coaching-institute website scaffold. Files created:

# `index.html` — Home page
# `classes.html` — Classes (4 items)
# `books.html` — Books (4 items)
# `team.html` — Our Team (4 members)
# `contact.html` — Contact form
# `assets/css/style.css` — Styles (responsive + background)
# `assets/js/main.js` — Mobile nav toggle

To open locally, open `index.html` in your browser. For quick local server (optional):

Windows PowerShell:
```
# from project folder
python -m http.server 8000
```

Then visit http://localhost:8000
